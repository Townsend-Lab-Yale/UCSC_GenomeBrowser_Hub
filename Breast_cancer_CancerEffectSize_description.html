TCGA Cancer Effect Size Track Hub

Cancer effect size quantifies natural selection on somatic mutations across 20 TCGA cancer cohorts. Each track displays Cancer Effect Size (CES) values from cancereffectsizeR - how much each mutation contributes to cancer cell proliferation in specific tumor types.

Display Conventions

Each subtrack shows CES scores transformed via f(x) = (10 Â· log(x)) - 10:

  • Positive score = Positive selection
  • Score = 1 = Neutral selection
  • Score = 0 = Negative selection
Reverse: 10((y+10)/10)

Search by gene_variant names (e.g., KIF1B_R1320W) in UCSC search box.

Methods

CES calculated by cancereffectsizeR: estimates neutral mutation rate per site using tissue-specific gene mutation rates + mutational signatures, then compares to observed rates [web:36].
